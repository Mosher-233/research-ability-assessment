# 研究能力评价系统演示脚本

## 1. 系统概述

研究能力评价系统是一个基于AI Agent的大学生研究能力评价平台，旨在帮助教师对学生的研究能力进行全面、客观的评估。系统通过收集学生的研究证据，利用AI技术进行能力推理，并生成详细的评价报告。

## 2. 演示环境

- **后端服务**：http://localhost:8080
- **前端服务**：http://localhost:3000
- **数据库**：PostgreSQL + Neo4j

## 3. 演示步骤

### 3.1 登录系统

1. 打开浏览器，访问前端服务地址：http://localhost:3000
2. 进入登录页面，输入以下登录信息：
   - 邮箱：admin@example.com
   - 密码：password123
3. 点击「登录」按钮
4. 进入系统仪表盘

### 3.2 任务管理

#### 3.2.1 创建任务

1. 点击左侧导航栏中的「任务管理」
2. 点击「创建任务」按钮
3. 填写任务信息：
   - 任务名称：研究方法课程作业
   - 任务描述：完成一项关于研究方法的课程作业，包括文献综述、研究设计、数据分析等内容
   - 课程ID：RESEARCH-2024
   - 开始日期：当前日期
   - 结束日期：30天后
4. 点击「提交」按钮
5. 查看新创建的任务

#### 3.2.2 分配任务

1. 在任务列表中找到刚创建的任务
2. 点击「分配任务」按钮
3. 选择学生（例如：张三、李四、王五）
4. 点击「确定」按钮
5. 查看任务的学生分配情况

### 3.3 证据管理

#### 3.3.1 提交证据

1. 点击左侧导航栏中的「证据管理」
2. 选择任务：研究方法课程作业
3. 选择学生：张三
4. 点击「提交证据」按钮
5. 填写证据信息：
   - 证据类型：论文
   - 证据内容：本文通过对研究方法的系统梳理，探讨了定量研究和定性研究的优缺点，并结合实际案例进行了分析。
   - 知识点名称：研究方法
   - 知识点级别：3
6. 点击「提交」按钮
7. 查看提交的证据

### 3.4 结果管理

#### 3.4.1 查看评估结果

1. 点击左侧导航栏中的「结果管理」
2. 选择任务：研究方法课程作业
3. 选择学生：张三
4. 点击「查看详情」按钮
5. 查看评估结果，包括总体评分、各维度评分和详细推理过程

### 3.5 报告管理

#### 3.5.1 生成报告

1. 点击左侧导航栏中的「报告管理」
2. 点击「生成报告」按钮
3. 填写报告信息：
   - 学生选择：张三
   - 任务选择：研究方法课程作业
   - 报告类型：详细报告
4. 点击「生成报告」按钮
5. 在报告列表中查看生成的报告

#### 3.5.2 查看报告详情

1. 在报告列表中找到刚生成的报告
2. 点击「查看」按钮
3. 查看报告详情，包括学生信息、任务信息、能力评估结果、详细分析和改进建议
4. 点击「下载报告」按钮，下载报告

### 3.6 学生操作演示

#### 3.6.1 学生登录

1. 打开新的浏览器标签页，访问前端服务地址：http://localhost:3000
2. 进入登录页面，输入以下登录信息：
   - 邮箱：student@example.com
   - 密码：password123
3. 点击「登录」按钮
4. 进入学生仪表盘

#### 3.6.2 学生查看任务

1. 点击左侧导航栏中的「任务管理」
2. 查看分配给自己的任务列表
3. 点击「查看详情」按钮，查看任务详细信息

#### 3.6.3 学生提交证据

1. 点击左侧导航栏中的「证据管理」
2. 选择任务：研究方法课程作业
3. 点击「提交证据」按钮
4. 填写证据信息
5. 点击「提交」按钮

#### 3.6.4 学生查看评估结果

1. 点击左侧导航栏中的「结果管理」
2. 查看自己的评估结果列表
3. 点击「查看详情」按钮，查看详细评估结果

## 4. 系统功能亮点

### 4.1 AI Agent 能力评估

- **智能证据分析**：AI Agent能够自动分析学生提交的证据，提取关键信息
- **多维度评估**：从文献综述、研究设计、数据分析、批判性思维、学术写作等多个维度进行评估
- **详细推理过程**：提供详细的评估推理过程，确保评估的透明度和可解释性
- **个性化反馈**：根据评估结果，生成个性化的改进建议

### 4.2 完整的任务管理流程

- **任务创建**：教师可以创建详细的研究任务
- **任务分配**：支持批量分配任务给多个学生
- **进度跟踪**：实时跟踪学生任务的完成进度
- **证据管理**：支持多种类型的证据提交和管理

### 4.3 强大的报告生成功能

- **详细报告**：生成包含详细评估结果和分析的报告
- **摘要报告**：生成简洁的评估摘要报告
- **可视化展示**：通过图表展示学生的能力分布
- **导出功能**：支持导出报告为PDF格式

### 4.4 用户友好的界面

- **响应式设计**：适配不同屏幕尺寸
- **直观的导航**：清晰的侧边栏导航
- **交互式操作**：丰富的交互元素和动画效果
- **实时反馈**：操作后立即显示反馈信息

## 5. 技术特点

- **前后端分离**：使用Vue 3 + TypeScript + Element Plus构建前端，Go + Gin构建后端
- **AI集成**：集成OpenAI API，使用GPT-3.5-turbo模型进行能力评估
- **知识图谱**：使用Neo4j构建知识图谱，支持知识关联和推理
- **数据安全**：使用JWT进行身份认证，确保API访问的安全性
- **可扩展性**：模块化设计，支持功能扩展和技术升级

## 6. 演示总结

通过本演示脚本，您可以了解研究能力评价系统的主要功能和使用流程，包括：

- 教师如何创建和分配任务
- 学生如何提交研究证据
- AI Agent如何评估学生的研究能力
- 系统如何生成详细的评价报告

系统的核心价值在于利用AI技术，为教师提供客观、全面的学生研究能力评估，同时为学生提供个性化的改进建议，帮助学生提高研究能力。
